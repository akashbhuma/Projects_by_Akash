AI Excuse Generator

An interactive AI-powered Excuse Generator built using Python and Gradio.
This project generates creative excuses, converts them into voice, allows saving favorites, and even generates images based on excuses — all through a simple web interface.

1.Features
2.Generate AI Excuses (text-based)
3.Text-to-Speech Excuses using ElevenLabs
4.Save & View Favorite Excuses
5.Generate Images from Excuses
6.Web UI using Gradio
7.Optional integration with Twilio for messaging

Tech Stack

Python 3
Gradio – Web interface
OpenAI API – Text generation
ElevenLabs API – Voice generation
Twilio API – Messaging (optional)

Working Process
User selects an excuse type
AI generates a creative excuse
Optional:
	Convert excuse to voice
	Save excuse to favorites
	Generate an image from the excuse
Output is shown instantly in the UI
